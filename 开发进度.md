## 📋 项目概述

基于 LangChain + RAG + Agent 架构的本地知识库系统，具有以下核心特性：

- **双模式LLM**: DeepSeek-V2-Lite 本地模型 (16B MoE, 4-bit量化) + 大模型API后备
- **完整RAG流水线**: 文档处理 → 向量化 → 检索 → 增强生成
- **中文环境优化**: 针对中英文混合文献的特殊处理
- **多界面支持**: Streamlit 可视化界面 + FastAPI REST API
- **MinerU集成**: 自动导入PDF转换的Markdown文档
- **元数据管理**: 自动提取标题、作者、摘要、关键词
- **图片关联**: 自动处理文档中的图片

## ⚡ 快速开始

```bash
# 1. 安装依赖
pip install -r requirements.txt

# 2. 配置环境变量
cp .env.example .env
# 编辑 .env 填入 API Key

# 3. 启动服务
start.bat
# 或
python src/api/main.py  # 终端1
streamlit run src/frontend/app.py  # 终端2

# 4. 访问
http://localhost:8501
```
