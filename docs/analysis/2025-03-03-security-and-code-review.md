## 具体修复方案

### 安全修复示例

#### 修复eval()风险
```python
# 当前问题代码
result = eval(expression, {"__builtins__": {}}, allowed_names)

# 修复方案: 使用simpleeval
from simpleeval import simple_eval

def safe_eval(expression, allowed_names):
    return simple_eval(
        expression,
        names=allowed_names,
        functions={},
        max_chars=1000,
        max_power=1000
    )
```

#### 修复路径遍历
```python
# 当前问题代码
full_path = Path(base_path) / file_path

# 修复方案
def safe_join(base_path, user_path):
    resolved = (Path(base_path) / user_path).resolve()
    base_resolved = Path(base_path).resolve()
    if not str(resolved).startswith(str(base_resolved)):
        raise ValueError("路径遍历攻击检测")
    return resolved
```
