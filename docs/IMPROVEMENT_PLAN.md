## 五、参考资源

### 主流RAG最佳实践（2024-2025）

1. **Advanced RAG架构**：Query变换 → 混合检索 → 重排序 → 生成
2. **混合检索**：向量检索 + BM25，RRF融合
3. **分块策略**：语义分块、页面级分块、Late Chunking
4. **重排序**：Cross-encoder (bge-reranker-v2-m3) 是SOTA
5. **评估框架**：RAGAS、TruLens (RAG Triad)
